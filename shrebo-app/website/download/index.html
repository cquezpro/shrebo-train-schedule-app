iPhone 4S, iPhone 5/5S, iPhone 6/6 Plus

Bitte öffnen Sie das TestFairy Email auf Ihrem iPhone. Darin finden Sie einen Link, um Ihr iPhone für cleverpendeln einmalig zu registrieren1.

Sie erhalten nach einigen Stunden ein zweites Email2 mit einem Link, um die cleverpendeln App ganz normal zu installieren.

Falls Sie bei der Erstinstallation Hilfe benötigen finden Sie dazu eine detaillierte Anleitung für die Erstinstallation für iPhone (PDF)

1Aufgrund technischer Vorgaben von Apple muss Ihr iPhone einmalig registriert werden, um teilzunehmen. Diese Registrierung dient rein technischen Zwecken und hat keinen Einfluss auf die Sicherheit der Daten auf Ihrem Telefon.
2Dieser Vorgang kann leider derzeit nur während den Bürozeiten erfolgen und kann daher einige Stunden dauern.
 

Smartphone mit Android Version 4.x

Bitte öffnen Sie das TestFairy Email auf Ihrem Android Smartphone. Darin ist ein Link enthalten, mit dem Sie die cleverpendeln App installieren können.

Falls Sie bei der Erstinstallation Hilfe benötigen finden Sie dazu eine detaillierte Anleitung zur Erstinstallation mit Android (PDF)
