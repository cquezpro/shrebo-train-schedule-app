shrebo @ bbookers
=================

Installation für Android
-------------------------
1. .apk Datei mit URL-Link oder per Dropbox Client auf Android Telephon kopieren
2. Die Installation sollte automatisch starten
3. Nach der Installation beider Datein gibt es zwei App-Icons: bbookers-staff und bbookers-customer
4. Wenn es nicht klappt, Apps bitte manuell loeschen und nochmals versuchen.

Unschönheiten, Bugs, Probleme? Bitte melden!
--------------------------------------------

So:
1. https://waffle.io/shrebo/bbookers öffnen
2. "Add Issue" klicken
3. Der neue Issue erscheint links oben gleich wieder. Auf die Zahl klicken. Es öffnet sich ein Popup
4. Beobachtungen schildern. Am liebsten so: 
   * Was habe ich bemerkt? 
   * Wie kann man das Problem reproduzieren? 
   * Wie sollte es sein?

   Beispiel: https://waffle.io/shrebo/bbookers/cards/544e802b7abdfe69daba1968

PS: Ich freue mich wenn ihr Probleme findet & meldet - so werden wir sie gemeinsam los. 

Aber bitte schreibt mir keine Emails mit Problemen - dann habe ich immer den doppelten Aufwand, weil ich
das in eine Todo Liste packen muss, damit es dann auch abgearbeitet wird... 

Deshalb meine Bitte: Fuer alle einfacher und übersichtlicher ist es, gleich einen Issue hier zu eröffnen:
https://waffle.io/shrebo/bbookers 

Danke!
